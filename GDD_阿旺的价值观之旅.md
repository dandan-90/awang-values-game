## 1. 设计支柱

### 1.1 四大支柱

| 支柱 | 描述 | 衡量标准 |
|------|------|----------|
| **寓教于乐** | 通过游戏机制自然传递企业价值观，不靠说教 | 玩家能说出每个宝物对应的关卡机制 |
| **可爱亲和** | 阿旺形象圆润温暖，界面友好，降低游戏门槛 | 非游戏玩家首次上手成功率 > 90% |
| **成就感驱动** | 每个宝物收集都有强烈的正面反馈 | 每次收集触发粒子效果 + 音效感知 |
| **双关变奏** | 两关玩法截然不同，保持新鲜感 | 第二关的留存率 ≥ 第一关的 70% |

### 1.2 反设计原则（我们不做的事）

- ❌ 不添加排行榜/竞技元素（聚焦内在动机）
- ❌ 不含广告或内购
- ❌ 不以数值膨胀驱动进度
- ❌ 不对玩家失败进行惩罚性设计

## 2. 世界观与叙事

### 2.1 故事背景

> 在三旺星球，一台小型的工业交换机日复一日吸收着周围的能量与灵感。某一天，它渐渐苏醒，拥有了生命——这就是**阿旺**。
>
> 阿旺外形圆润可爱，肚子上还保留着几个小巧的交换机接口，仿佛随时准备连接一切可能。
>
> 他的使命是收集散落在星球上的 **8 枚价值观宝物**，凝聚三旺精神，带领三旺通信走向 **年营收 10 亿** 的辉煌。

### 2.2 叙事结构

```
序幕（菜单） → 第一关过场 → 横版闯关(4宝物) → 第一关结算
                                           ↓
结局（10亿）  ← 第二关结算 ← 肉鸽模式(4宝物) ← 第二关过场
```

### 2.3 情感曲线

```
玩家情绪
  ▲           ┌── 肉鸽探索的紧张感
  │      ┌── 横版闯关│       ┌── 通关高潮
  │     /    流畅感   │      /
  │    /              │     /
  │───┘               └────┘──────→ 时间
  菜单                第一关完成    最终胜利
```

## 3. 核心玩法循环

### 3.1 Moment-to-Moment（0-30秒）

| 关卡 | 动作 | 反馈 | 奖励 |
|------|------|------|------|
| 第一关 | 移动/跳跃/踩踏敌人 | 视觉：阿旺弹性形变 + 粒子特效 | 进度条推进 |
| 第二关 | 移动/攻击/探索房间 | 视觉：房间变色/敌人消失 | 房间清除标记 |

### 3.2 Session Loop（5-15分钟）

- **第一关目标**: 从起点跑到终点，沿途收集 4 个宝物
- **第一关张力**: 敌人、陷阱、掉落风险
- **第二关目标**: 探索随机生成的地牢，收集 4 个宝物 + 击败 BOSS
- **第二关张力**: 随机房间、永久死亡（关卡内）、生命值管理

### 3.3 Long-Term（完整通关约20-30分钟）

- **进度锚点**: 第一关和第二关之间的过场结算
- **最终目标**: 收集全部 8 个宝物 → 达成年营收 10 亿
- **重玩价值**: 第二关房间随机生成

## 4. 第一关：横版闯关

### 4.1 玩法概述

**类型**: 2D 横版跳跃平台游戏 (Platformer)
**视野**: 水平滚动，摄像机平滑跟随
**长度**: ~5000px 关卡宽度（约 3-5 分钟流程）

### 4.2 关卡结构

```
[起点] ── 基础教学区 ── [洞察需求] ── 跳跃挑战区 ── [创造价值] ── 敌人密集区 ──
  [合作共赢] ── 快速反应区 ── [敏捷响应] ── 最终冲刺 ── [终点旗帜]
```

| 区域 | 长度 | 主要机制 | 宝物 |
|------|------|----------|------|
| Zone 1: 教学区 | 0-800px | 基础移动、跳跃 | — |
| Zone 2: 洞察区 | 801-1200px | 首次宝物收集 | 洞察需求 |
| Zone 3: 创造区 | 1201-1800px | 跳跃 + 敌人引入 | 创造价值 |
| Zone 4: 合作区 | 1801-2700px | 多平台跳跃、敌人巡逻 | 合作共赢 |
| Zone 5: 敏捷区 | 2701-3600px | 快速跳跃、连续平台 | 敏捷响应 |
| Zone 6: 冲刺区 | 3601-5000px | 最终挑战、终点 | — |

### 4.3 机制规格

#### 4.3.1 玩家移动

| 参数 | 值 | 调优说明 |
|------|-----|----------|
| 水平速度 | 280 px/s | `[PLACEHOLDER]` 需测试手感 |
| 跳跃力 | -520 px/s | 对应约 2.5 个玩家身高 |
| 重力 | 1200 px/s² | 产生约 0.8 秒的跳跃时长 |
| 最大下落速度 | 800 px/s | 防止无限加速 |
| 空中控制 | 100% | 新手友好，允许空中调整 |

**玩家体验**: 跳跃应该感觉"有弹性但可控"，阿旺落地时有轻微 squash 形变

#### 4.3.2 敌人

| 类型 | 血量 | 速度 | 行为 |
|------|------|------|------|
| 数据噪音（小怪） | 1（踩踏击杀） | 60 px/s | 水平巡逻 |

- **击败方式**: 从上方踩踏（玩家 vy > 0 且在敌人头顶）
- **碰撞伤害**: 侧面碰撞触发无敌帧 1.5s
- **反馈**: 击败粒子（紫色）+ 轻微屏幕震动

#### 4.3.3 陷阱

| 类型 | 伤害 | 反馈 |
|------|------|------|
| 尖刺 | 触发无敌帧 + 弹飞 | 红色粒子 + 震屏 |

### 4.4 宝物收集

- **放置策略**: 每个宝物位于该区域的核心位置，需要一定技巧才能到达
- **收集反馈**: 20 个彩色粒子爆发 + 宝物名称飞字动画
- **收集条件**: 无需所有宝物即可通关，但完整收集才能解锁第二关叙事

### 4.5 边界情况处理

| 情况 | 处理 |
|------|------|
| 玩家掉出屏幕下方 | 重置到起点附近，保留已收集宝物 |
| 连续碰撞多个敌人 | 无敌帧保护，不会连续受伤 |
| 宝物重复碰撞 | 已收集宝物不可重复触发 |

## 5. 第二关：肉鸽模式

### 5.1 玩法概述

**类型**: 房间制 Roguelike 地牢探索
**地图**: 5×4 网格房间（20 间）
**视角**: 俯视 (Top-down)
**核心循环**: 探索房间 → 击败敌人 → 收集宝物 → 击败 BOSS

### 5.2 房间类型

| 类型 | 数量 | 特性 | 占领条件 |
|------|------|------|----------|
| 战斗房 | 9 | 1-3 个普通敌人 | 击败所有敌人 |
| 宝物房 | 4 | 各含 1 个价值观宝物 | 进入即取 |
| 治疗房 | 1 | 缓慢回血 (0.5 HP/s) | 无需占领 |
| BOSS 房 | 1 | 5 HP 的 BOSS | 击败 BOSS |
| 空房 | 5 | 安全通行区 | 无需占领 |

### 5.3 房间分布（预设）

```
┌─────────┬─────────┬─────────┬─────────┬─────────┐
│ 主动担当 │  战斗房  │  战斗房  │  战斗房  │ 实干真做 │  Row 0
│ 💎宝物   │   ⚔️    │   ⚔️    │   ⚔️    │ 💎宝物   │
├─────────┼─────────┼─────────┼─────────┼─────────┤
│  战斗房  │  战斗房  │  战斗房  │  战斗房  │  战斗房  │  Row 1
│   ⚔️    │   ⚔️    │   ⚔️    │   ⚔️    │   ⚔️    │
├─────────┼─────────┼─────────┼─────────┼─────────┤
│  空房   │  战斗房  │  治疗房  │  战斗房  │  空房   │  Row 2
│   ·     │   ⚔️    │   ❤️    │   ⚔️    │   ·     │
├─────────┼─────────┼─────────┼─────────┼─────────┤
│ 持续创新 │  战斗房  │  起点房  │  战斗房  │ 结果导向 │  Row 3
│ 💎宝物   │   ⚔️    │   🏠    │   ⚔️    │ 💎宝物   │
└─────────┴─────────┴─────────┴─────────┴─────────┘
```

**设计理由**: 
- 宝物放在四角 → 迫使玩家探索全图
- BOSS 在顶部中央 → 心理上需要充分准备
- 治疗房在中央偏下 → 战略休息点

### 5.4 BOSS 设计

| 参数 | 值 | 调优说明 |
|------|-----|----------|
| 血量 | 5 HP | `[PLACEHOLDER]` 每次攻击 1 点伤害 |
| 移动速度 | 40 px/s | 比普通敌人慢，但体型更大 |
| 攻击冷却 | 0.3s | 与玩家攻击冷却相同 |
| 视觉 | 红色 + 金色皇冠 | 明显区分于普通敌人 |

**玩家体验**: BOSS 应该让玩家感到"有威胁但可战胜"。5 次攻击击败，需要玩家在攻击间隔中移动躲避。

### 5.5 肉鸽要素设计

| 要素 | 实现 |
|------|------|
| **随机生成** | 敌人位置、数量、移动模式随机 |
| **永久死亡** | HP 归零后本关重置（保留地图结构，重置敌人） |
| **可重试** | 死亡 1.5s 后自动重生，无惩罚 |
| **进度保留** | 已收集宝物在重试后保留 |

### 5.6 房间转换

- **触发方式**: 玩家走到房间边缘的门位置
- **转换冷却**: 0.15s，防止连续切换
- **新房间入口位置**: 从进入的门对面出现

## 8. UI/UX 设计

### 8.1 界面状态机

```
MENU → LEVEL1_INTRO → LEVEL1 → LEVEL1_COMPLETE
                                     ↓
                                  LEVEL2_INTRO → LEVEL2 → LEVEL2_COMPLETE
                                                              ↓
                                                           WINNING → MENU
```

### 8.2 HUD 设计

#### 8.2.1 第一关 HUD
- `[左上]` 进度条（200×12 px）+ 百分比
- `[右上]` 关卡名称 + 宝物收集计数 (x/4)
- `[右下]` 已收集宝物列表（图标 + 名称）
- `[左下]` 操作提示

#### 8.2.2 第二关 HUD
- `[左上]` HP 血条（160×16 px），颜色随血量变化（绿→橙→红）
- `[右上]` 关卡名称 + 宝物收集计数
- `[右下]` 已收集宝物列表
- `[中下]` 消息弹窗（房间清除/宝物获取/BOSS击败），2.5s 自动消失
- `[左下]` 操作提示

### 8.3 过场动画

| 过场 | 时长 | 内容 |
|------|------|------|
| 关卡介绍 | 等待按键 | 关卡名 + 宝物预览 + 操作说明 |
| 关卡完成 | 等待按键 | 已收集宝物 ✓ 列表 |
| 最终胜利 | 持续展示 | 全部 8 宝物 + "10亿" 大数字揭示 |

### 8.4 操作映射

| 操作 | 键盘 | 功能 |
|------|------|------|
| 移动（横版） | ← → / A D | 水平移动 |
| 跳跃 | ↑ / W / 空格 | 跳跃 |
| 移动（肉鸽） | WASD / 方向键 | 四方向移动 |
| 攻击（肉鸽） | 空格 / J | 近战攻击（50px 范围） |
| 确认 | 空格 / 回车 / 点击 | 菜单/过场推进 |

## 9. 经济与数值平衡

### 9.1 第一关难度曲线

| 区域 | 平台密度 | 敌人数量 | 陷阱数量 | 预计死亡风险 |
|------|----------|----------|----------|-------------|
| Zone 1 | 低 | 0 | 0 | 0% |
| Zone 2 | 低 | 0 | 1 | 5% |
| Zone 3 | 中 | 1 | 1 | 15% |
| Zone 4 | 中 | 2 | 1 | 25% |
| Zone 5 | 高 | 1 | 1 | 30% |
| Zone 6 | 中 | 1 | 1 | 20%（有经验后降低） |

### 9.2 第二关难度曲线

| 指标 | 值 | 调优说明 |
|------|-----|----------|
| 玩家 HP | 5 | `[PLACEHOLDER]` 测试通关率 |
| 普通敌人 HP | 2 | 需要 2 次攻击 |
| BOSS HP | 5 | 需要 5 次攻击 |
| 普通敌人攻击 | 1 伤害 | 触碰即伤 |
| 敌人数量/战斗房 | 1-3 | `[PLACEHOLDER]` 随机 |
| 无敌帧 | 1.0s | 防止连续受伤 |
| 治疗房回血 | 0.5 HP/s | 约 10 秒回满 |
| BOSS 移动速度 | 40 px/s | 比普通敌人慢 20% |

### 9.3 平衡目标

| 指标 | 目标值 | 测量方式 |
|------|--------|----------|
| 第一关首次通关率 | > 80% | 第一关终点到达率 |
| 第一关完整收集率 | > 50% | 4/4 宝物收集率 |
| 第二关首次通关率 | > 50% | BOSS 击败 + 4/4 宝物 |
| 第二关重试后通关率 | > 90% | 带着知识重试 |
| 平均通关时长 | 15-25 分钟 | 计时统计 |
